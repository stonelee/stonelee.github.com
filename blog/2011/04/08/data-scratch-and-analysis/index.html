1.分析网络包——Wireshark

  • ubuntu下需要sudo wireshark启动
  • Capture-interfaces:Start有ip的Device
  • 本机网络包无法抓取
  • filter中填写:
    • http and ip.src==192.168.1.103只显示本机发送的http协议包
    • http and ip.addr==192.168.1.103只显示本机发送和接收的http协议包
